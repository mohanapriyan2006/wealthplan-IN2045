## ✨ Overview
[WealthPlan IN2045](https://mohanapriyan2006.github.io/wealthplan-IN2045/) is a **professional SaaS-style investment dashboard** designed with a **purple gradient UI** and **smooth animations**. It helps investors track portfolios, analyze performance, and gain actionable insights with elegance.

---

## 🚀 Features
- 📊 **Portfolio Dashboard** – Interactive cards with live updates  
- 📈 **Charts & Graphs** – Performance, allocation, and risk analysis (powered by Chart.js/D3.js)  
- 📉 **Analytics Widgets** – ROI, CAGR, Sharpe ratio, volatility indicators  
- 📰 **News & Insights** – Auto-scroll ticker with fade/slide animations  
- 🔍 **Watchlist & Comparison Tool** – Side-by-side equity/fund comparisons with animated transitions  
- 🎨 **Purple Gradient UI** – Modern SaaS design with hover glows, ripple effects, and parallax scroll  

---

## 🛠 Installation
Clone the repo and install dependencies:

```bash
git clone https://github.com/mohanapriyan2006/wealthplan-IN2045.git
cd wealthplan-IN2045
npm install
npm start
```

---

## 📖 Usage
1. Launch the app locally.  
2. Add assets to your portfolio.  
3. Explore charts and analytics.  
4. Compare equities and track insights.

## 🧑‍💻 Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Visualizations:** Chart.js / D3.js  
- **Animations:** CSS keyframes, IntersectionObserver  
- **Backend (optional):** Spring Boot / Node.js
